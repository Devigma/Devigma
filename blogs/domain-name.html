DNS (DOMAIN NAME SYSTEM)

In order to understand how domain name works,users will have to know about DNS. Domain Name System is a decentralized system that stores information about which domains are connected to a certain IP address. When a user enters your domain name in their browser, it checks the DNS for the corresponding IP address and direct the user to the right page.

TLD (TOP LEVEL DOMAIN)
